## General info
This project is TodoList App ,in which a user can maintain list by inserting , deleting and updating current list items
	
## Technologies
Project is created with:
* React.js
* Material UI
* LocalStorage Hook
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../todo_list
$ npm install
$ npm start
```
